# 🎀 Happy 21st Birthday, Preety

A breathtakingly beautiful 4-page birthday surprise website.

## Pages

| Page | Description |
|------|-------------|
| 🏠 Home | Grand cinematic hero with shimmer name, falling petals, confetti |
| 🌸 Memories | Friendship timeline — Kovil, CSC Journey, Falooda |
| 📷 Gallery | Polaroid-style photo frames you can fill with real photos |
| 💌 Letter | Envelope reveal with a secret handwritten letter |

## Features

- Animated falling rose petals & cherry blossoms (all pages)
- Glowing shimmer text with gold gradient
- Confetti burst on load + envelope open
- Custom pink heart cursor
- Bottom pill navigation + side dot nav
- Scroll-triggered entrance animations
- Photo upload for all 5 gallery frames
- 3D envelope flip animation → letter reveal
- Background music toggle button
- Keyboard arrow + touch swipe navigation
- Fully mobile responsive

## Local Dev

```bash
npx serve . -p 3000
# Open http://localhost:3000
```

## Deploy to Vercel

```bash
# 1. Init git
git init
git add .
git commit -m "🎀 Happy Birthday Preety"

# 2. Push to GitHub
gh repo create preety-birthday --public --source=. --push

# 3. Deploy
npx vercel --prod
```
